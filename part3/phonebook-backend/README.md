# Phonebook Backend

Phonebook application backend built with Express.

## Live Demo

[https://phonebook-backend.onrender.com/api/persons]

## API Endpoints

- `GET /api/persons` - Get all contacts
- `GET /api/persons/:id` - Get a single contact
- `POST /api/persons` - Add a new contact
- `DELETE /api/persons/:id` - Delete a contact

## Technologies

- Node.js
- Express
- Morgan (logging)
- CORS

## Local Development

```bash
npm install
npm run dev
```
